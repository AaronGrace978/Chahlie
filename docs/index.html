🎮 Steam Deck (recommended)

Chahlie runs your Deck — launch games, check battery, set volume, organize Downloads, or run shell commands. The v2.7.3 UI is tuned for the Deck's 1280×800 screen with a big chat area and touch-friendly buttons.

Download v2.7.3 Deck build Get free Ollama API key All releases 
cd ~/Downloads
curl -L -O https://github.com/AaronGrace978/Chahlie/releases/download/v2.7.3/chahlie-deck-2.7.3-linux-x86_64.tar.gz
tar -xzf chahlie-deck-2.7.3-linux-x86_64.tar.gz
cd chahlie-deck-2.7.3-linux
chmod +x START-CHAHLIE.sh
./START-CHAHLIE.sh
Launch tip: Double-click Start Chahlie.desktop — not the .sh file (Kate opens that). Gold box at the bottom = where you type. Press F7 if the keyboard won't focus.
Try saying: "launch Steam" · "how's my battery?" · "set volume to 50" · "organize my Downloads"

🔌 Llama proxy (share Chahlie with friends)

Run a local OpenAI-compatible proxy so others can point Chahlie at your Ollama instance — great for LAN parties or a home server with a GPU.

# On the machine running Ollama:
ollama serve
ollama pull qwen3:8b

# Start the Chahlie proxy (forwards to Ollama):
python -m chahlie.llama_proxy --port 11435

# On a friend's Deck or PC:
CHAHLIE_BACKEND=openai-compatible
OPENAI_COMPATIBLE_URL=http://YOUR-IP:11435/v1
OPENAI_COMPATIBLE_MODEL=qwen3:8b
Optional: set LLAMA_PROXY_API_KEY on the server and OPENAI_COMPATIBLE_API_KEY on clients so random people can't burn your GPU cycles.

🔧 Troubleshooting

  • No module named chahlie — use v2.7.1+ or run: ~/.local/share/chahlie/venv/bin/pip install -e ~/Downloads/chahlie-deck-*-linux
  • UI crash on launch — update to v2.7.2+ (fixed Textual timer bug)
  • Chat errors / NoneType result — update to v2.7.3
  • Kate opens instead of running — double-click Start Chahlie.desktop
  • Bad API key — type /key in the chat box
Need the full guide? See STEAM_DECK.md on GitHub.
