Chahlie Steam Deck Edition — Install Guide
==========================================

DOWNLOAD → EXTRACT → INSTALL → RUN

1. Download the release file:
   chahlie-deck-VERSION-linux-x86_64.tar.gz
   from GitHub Releases:
   https://github.com/AaronGrace978/Chahlie/releases

2. Open Konsole (Desktop Mode) and extract:

   cd ~/Downloads
   tar -xzf chahlie-deck-*-linux-x86_64.tar.gz
   cd chahlie-deck-*-linux

3. Run the installer (one time):

   ./install.sh

   This installs Python deps WITHOUT compiling PyAudio.
   Voice uses your Deck's built-in mic tools (pw-record + espeak-ng).

4. Add your Ollama API key:

   nano ~/.local/share/chahlie/.env
   # set OLLAMA_API_KEY=your-key-here

5. Launch Chahlie:

   chahlie-deck

   Or double-click "Chahlie (Steam Deck)" in your app menu.

6. (Optional) Add to Steam Gaming Mode:
   Steam → Add Game → Non-Steam Game → chahlie-deck

TROUBLESHOOTING
---------------
- Mic not working:  sudo pacman -S pipewire-pulse
- No voice out:     sudo pacman -S espeak-ng
- Install failed:   make sure you're in Desktop Mode with internet

Need help? See STEAM_DECK.md in this folder.
