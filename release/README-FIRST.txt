FIRST TIME ONLY
───────────────
1. Download from:  github.com/AaronGrace978/Chahlie/releases
   File:  chahlie-deck-*-linux-x86_64.tar.gz

2. Right-click the download → Extract Here

3. Open the extracted folder in Dolphin (file manager)

4. Find:  "Start Chahlie.desktop"

5. RIGHT-CLICK it → Properties → Permissions
   → check "Is executable" → OK

6. RIGHT-CLICK it again → "Allow Launching" (or "Trust and Launch")

7. DOUBLE-CLICK "Start Chahlie.desktop"

8. A terminal opens, installs once (~1 min), then the CHAHLIE UI appears.
   Paste your free API key when asked.
   Get key:  ollama.com/settings/keys


STILL STUCK? COPY-PASTE THIS IN KONSOLE (the terminal app)
──────────────────────────────────────────────────────────
cd ~/Downloads/chahlie-deck-*-linux && chmod +x START-CHAHLIE.sh && ./START-CHAHLIE.sh


AFTER FIRST RUN
───────────────
Use the "Start Chahlie" icon on your Desktop.
